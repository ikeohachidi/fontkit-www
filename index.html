Font Import

Get up and running by easily importing a font at the click of a button and following three easy steps.

  • Click the import font button
  • Select the fonts you wish to import
  • Click Done

Collections

Collections allow a way to mix and match fonts however you like. A collection provides a URL which can be used in your sites. The URL never changes even if the font changes, so you never have to worry about going back and forth with copy pasting URLs.

Secure

Uploaded a premium font to a collection you don’t want someone to easily steal from your website. Easy solution, add authorized URL’s the collection will only work with.
